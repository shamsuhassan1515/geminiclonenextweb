# Gemini Clone

Gemini Clone Web Application - AI Chat Interface

## 📋 项目信息

- **项目名称**: Gemini Clone
- **版本**: 0.01
- **作者**: shamsuhassan1515
- **类型**: Node.js Web 应用
- **端口**: 1002

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 🌐 访问地址

- **开发服务器**: http://localhost:1002
- **Gemini 页面**: http://localhost:1002/#/gemini
- **Chat 页面**: http://localhost:1002/#/chat

## 📦 技术栈

- **前端框架**: Vue 3
- **开发语言**: TypeScript
- **构建工具**: Vite
- **UI 组件库**: Naive UI
- **状态管理**: Pinia
- **路由**: Vue Router

## 🔧 开发环境

- **Node.js**: >= 18.0.0
- **npm**: >= 8.0.0
